# Bar Stazione Cefalù

Sito vetrina statico realizzato per presentare Bar Stazione Cefalù, attività situata in Piazza Vittime 16 Marzo 1978, 90015 Cefalù (PA), con focus su servizi, orari, contatti e posizione.

## Obiettivo del progetto

Il sito è pensato per comunicare in modo chiaro e moderno:
- i servizi disponibili
- gli orari di apertura
- i contatti
- la posizione
- la natura pratica e accogliente dell’attività

## Dati attività

- Nome: Bar Stazione Cefalù

## Avvio locale

Apri `index.html` nel browser oppure usa una semplice estensione Live Server.

## Come pubblicare su GitHub

1. Crea una nuova repository chiamata `bar-stazione-cefalu`
2. Carica i file del progetto
3. Esegui questi comandi da terminale:

```bash
git init
git add .
git commit -m "Initial commit - Bar Stazione Cefalù website"
git branch -M main
git remote add origin https://github.com/TUO-USERNAME/bar-stazione-cefalu.git
git push -u origin main
```

## Come pubblicare su Vercel

1. Accedi a Vercel
2. Clicca su **Add New Project**
3. Importa la repository GitHub `bar-stazione-cefalu`
4. Lascia le impostazioni standard per sito statico
5. Clicca **Deploy**

Vercel supporta l’importazione di repository GitHub e le distribuzioni automatiche ad ogni nuovo push sul branch collegato.

## Aggiornamenti rapidi

Per modificare i contenuti principali aggiorna:
- `index.html` per testi, sezioni, contatti e struttura
- `style.css` per colori, font, spaziature e responsive
- `script.js` per interazioni leggere

## Dati facili da aggiornare

Aggiorna questi elementi quando necessario:
- numero di telefono
- orari
- indirizzo
- servizi
- claim finale
- meta title e meta description
