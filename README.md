# Bar Stazione Cefalù - Web Agency Premium

Sito web statico *one-page* di livello premium, riprogettato con un'estetica moderna, raffinata e accattivante. Ottimizzato per dispositivi mobile e curato nei minimi dettagli di tipografia e animazioni.

## 🌟 Novità della versione Premium
- **Design Unico**: Interfaccia che si discosta dai classici template, con un focus su layout asimmetrici, grid moderne e un forte ritmo verticale.
- **Micro-animazioni**: Inseriti fade-in morbidi allo scroll (`IntersectionObserver`) e hover effect eleganti sulle card.
- **Tipografia**: Combinazione di `Playfair Display` (classico/elegante) e `DM Sans` (moderno/pulito) per la massima leggibilità e sofisticazione.
- **Palette Mediterranea**: Tonalità calde (sabbia, avorio) bilanciate da un elegante verde petrolio (teal) e testo antracite caldo.
- **Footer Arricchito**: Inclusa firma "Realizzato con ❤️ da PromoWeb" con animazione pulsante e link standard a Privacy e Cookie Policy.

## 🛠 Stack Tecnologico
- **HTML5**: Semantico e accessibile.
- **CSS3 Vanilla**: Uso avanzato di CSS Variables, Flexbox, CSS Grid (layout asimmetrico per desktop) e funzioni matematiche (`clamp` per tipografia fluida).
- **JavaScript (Vanilla)**: Logica per menu mobile, sticky header e `IntersectionObserver` per le animazioni.

## 📂 Struttura File
- `index.html`: La struttura della pagina e i contenuti testuali.
- `style.css`: Stile, animazioni keyframe, e media queries.
- `script.js`: Interazioni lato client.
- `README.md`: Documentazione del progetto.

## 🌐 Deploy
Ideale per Vercel, GitHub Pages o Netlify. Nessun processo di build necessario.
